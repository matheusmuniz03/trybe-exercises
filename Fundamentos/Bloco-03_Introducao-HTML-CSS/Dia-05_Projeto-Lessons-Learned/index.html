Nas minhas três primeiras na Trybe, aprendi muita coisa! Começamos já na primeira semana vendo o conteúdo sobre Unix & Bash, lá tive meu primeiro contato com o terminal do Linux e seus principais comandos. Em seguida, botamos a "mão na massa" e, em insistentes horas de estudo, comecei a entender mais sobre Git e GitHub, criando meu primeiro repositório no GitHub. Agora, vimos o conteúdo sobre a linguagem de marcação HTML e a linguagem de estilo CSS e estou muito empolgado em dominar essas linguagens.

Durante meus estudos sobre Git e GitHub, me deparei com alguns obstáculos. No começo, tive que criar e recriar pelo menos 5 vezes meu repositório, mas percebi que, trabalhando duro, fixei completamente o conteúdo.
